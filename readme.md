To run the CIT COURSEHUB Web APP
---------------------------------------------------------------------------------------------------------------
1. Make sure you have Flask installed. You can install Flask by running the following command in your terminal or command prompt:
```
pip install flask
```
2. Run main.py
```
py main.py
```
